# UML Now MCP

An MCP server around the UML Now API. An important component of [unibot-v2](unibot-v2.umlcloudcomputing.org).

Allows models capable for MCP tool calling to query against the UML Now API.

Written in Python. Uses [Docker](https://github.com/docker) and [MCP](https://modelcontextprotocol.io/docs/getting-started/intro). [MCPInspector](https://modelcontextprotocol.io/docs/tools/inspector) used for development.

Source code available on [GitHub](https://github.com/UMLCloudComputing/uml-now-mcp.git)
Published container image available [here](https://github.com/UMLCloudComputing/uml-now-mcp/pkgs/container/uml-now-mcp).
